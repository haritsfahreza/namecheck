# namecheck (`nck`)
Check your name idea availability with CLI

## Installation
```sh
$ go get -u github.com/uwuh/namecheck/cmd/nck//happy hacking
```

## Usage
```sh
$ nck help
Usage of nck:
  -help
        show this message
  -name string
        name idea that you want to check

```
```sh
$ nck -name uwuh
```
